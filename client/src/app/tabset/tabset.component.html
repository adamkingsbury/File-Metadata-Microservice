This file metadata microservice allows a user to upload a file to a server. The file will then be analysed to determine the height and width properties of the file. The server will send a JSON response with these properties included.

The microservice will also store the historical record of each request made to the server, including the filename, date and time of the request and the dimesions that were determined

The service has been extened beyond the original project requirements to include a demonstration interface and the capability to store the uploaded images files for viewing as part of history requests.
